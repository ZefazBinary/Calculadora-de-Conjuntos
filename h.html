Conjuntos

Em matemática, um conjunto é uma coleção bem definida de elementos ou objetos. Esses elementos podem ser números, letras, pessoas, frutas,etc., e são agrupados por alguma característica comum.


Exemplo: O conjunto das vogais da língua portuguesa pode ser representado assim: A = {a, e, i, o, u}

Notações importantes: Usa-se chaves { } para indicar os elementos do conjunto. Os elementos são separados por vírgulas. Um elemento não se repete dentro de um conjunto. A ordem dos elementos não altera o conjunto.

Tipos de Conjuntos

Conjunto Unitário: contém apenas um elemento. Ex: A = {7}

Conjunto Vazio: não possui elementos. Ex: B = { } ou B = ∅

Conjunto Universo: representa todos os elementos possíveis num contexto. Ex: U = {1, 2, 3, ..., 10}

Conjuntos Iguais: dois conjuntos com os mesmos elementos. Ex: C = {a, b}, D = {b, a}
